Early-2000s-Music-Player A retro-inspired mini music player built with HTML, CSS, and JavaScript that showcases tracks from early 2000s hip-hop.
The project features album artwork, play/pause controls, track navigation, and a sleek UI designed to deliver a nostalgic music listening experience in the browser.

Tech Stack: HTML | CSS | JavaScript

Made this while bored and decided to turn it into a simple nostalgic music player ;)
